Pocket Hero's Style Guide

Here's a simple style guide designed in order to help future Pocket Hero designers understand how to design interfaces for the app. Pocket Hero's style follows a simple design philosophy: ✨ "The simpler the better" ✨

Before designing, make sure to run these few commands in order to keep things going

1. Install Angular CLI: Make sure to add the -g instance so it's a global installation! if you don't, the path won't recognize the ng command and it won't let you build and deploy the project any further.


				npm install -g @angular/cli


2. Install the project dependencies: Make sure to run this command in the project's root directory (pocket-hero)




				npm i


3. Build and deploy the project.



In order to build it, run this command in the same directory


				ng b

And deploy it the serve command


				ng s
